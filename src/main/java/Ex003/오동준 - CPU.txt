CPU란

    CPU는 컴퓨터의 두뇌라고 생각하자  CPU는 많은 처리 장치 중 하나지만 가장 중요한 구성 요소이다.

    컴퓨터를 통한 연산과 수행 그리고 프로그램을 실행하는 컴퓨터의 중요부분


CPU는 출력 전달하기 전에 컴퓨터의 RAM에서 입력을 가져와 명령문을 이해하고 처리함


CPU는 어떻게 동작할까

세단계 분류 된다
    1. 데이터를 가져오기
    2. 디코딩
    3. 실행

1. 데이터를 가져오기
    - 데이터는 이진수로 표시되며 RAM 에서 CPU로 전달됨

2. 디코딩
    - 일단 명령을 가져와 IR에 저장하면  CPU는 명령 해독기라는 회로로 명령을 전달합니다.
    - 명령어는 CPU의 다른 부분으로 전달하여 작동을 위해 전달되는 신호로 변환됩니다.
3. 실행
 - 마지막 단계에서 명령문은 완료될 CPU의 관련 부분으로 전송됩니다.
